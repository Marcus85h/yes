# 聊天系统 (Chat)

聊天系统是视频交友应用的核心沟通模块，提供文字聊天、多媒体消息、群组聊天、私聊等功能。

## 🏗️ 系统架构

### 核心模型

1. **ChatRoom** - 聊天房间
   - 房间基础信息（房间ID、名称、描述）
   - 房间类型（私聊、群聊、房间聊天、系统消息）
   - 房间设置（私密、最大人数、权限控制）
   - 统计信息（消息数量、最后消息时间）

2. **ChatParticipant** - 聊天参与者
   - 参与信息（角色、状态、权限）
   - 权限控制（发送消息、邀请、管理）
   - 参与统计（消息数量、活跃时间）
   - 状态管理（活跃、禁言、封禁、离开）

3. **Message** - 聊天消息
   - 消息内容（文本、图片、视频、音频、文件）
   - 消息状态（发送、送达、已读、失败、删除）
   - 媒体信息（URL、路径、大小、时长）
   - 消息属性（编辑、回复、点赞）

4. **MessageRead** - 消息已读记录
   - 已读用户和时间记录
   - 支持消息已读统计

5. **MessageLike** - 消息点赞
   - 点赞用户和时间记录
   - 支持消息点赞统计

6. **ChatInvitation** - 聊天邀请
   - 邀请管理（邀请者、被邀请者）
   - 邀请状态（待处理、已接受、已拒绝、已过期）
   - 邀请消息和响应时间

7. **ChatNotification** - 聊天通知
   - 通知类型（新消息、提及、邀请、系统通知）
   - 通知状态（未读、已读、已归档）
   - 推送和邮件通知设置

## 📡 API接口

### 聊天房间
```
GET/POST /api/v1/chat/rooms/              # 房间列表/创建
GET/PUT/DELETE /api/v1/chat/rooms/{id}/   # 房间详情/更新/删除
POST /api/v1/chat/rooms/{id}/join/        # 加入房间
POST /api/v1/chat/rooms/{id}/leave/       # 离开房间
GET /api/v1/chat/rooms/stats/             # 聊天统计
```

### 消息管理
```
GET/POST /api/v1/chat/messages/           # 消息列表/发送
GET/PUT/DELETE /api/v1/chat/messages/{id}/ # 消息详情/更新/删除
POST /api/v1/chat/messages/{id}/mark_read/ # 标记已读
POST /api/v1/chat/messages/{id}/like/     # 点赞消息
POST /api/v1/chat/messages/{id}/unlike/   # 取消点赞
DELETE /api/v1/chat/messages/{id}/delete_message/ # 删除消息
```

### 聊天邀请
```
POST/GET /api/v1/chat/invitations/        # 发送/获取邀请
POST /api/v1/chat/invitations/{id}/accept/ # 接受邀请
POST /api/v1/chat/invitations/{id}/reject/ # 拒绝邀请
```

### 聊天通知
```
GET/POST /api/v1/chat/notifications/      # 获取/标记通知
```

### 其他接口
```
GET /api/v1/chat/my-chats/                # 我的聊天记录
POST /api/v1/chat/message-action/         # 消息操作
GET /api/v1/chat/rooms/{room_id}/messages/ # 房间消息
```
